## 🎯 About IDIOT Token

IDIOT Token is a meme cryptocurrency built on Base (Coinbase Layer 2) that combines humor with honest tokenomics. We're proof that you can be smart about having dumb fun in crypto.

### Key Features

- ✅ **Zero Taxes** - 0% buy, 0% sell, permanently
- ✅ **Base Network** - Fast and cheap transactions
- ✅ **Transparent** - All allocations public and verifiable
- ✅ **Community-First** - 70% of supply for community
- ✅ **Renounced** - No admin keys, truly decentralized
- ✅ **Liquidity Locked** - Your investment is safe
