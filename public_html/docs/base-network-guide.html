What Makes Base Special?

Base is designed to be the bridge between traditional finance and decentralized finance (DeFi).

  • Growing Ecosystem: 100+ dApps and growing

Adding Base to MetaMask

Follow these steps to add Base network to your MetaMask wallet:

1 Open MetaMask and click the network dropdown at the top
2 Click "Add Network" at the bottom of the dropdown
3 Enter Base Network Details:
  • Network Name: Base
  • RPC URL: https://mainnet.base.org
  • Chain ID: 8453
  • Currency Symbol: ETH
  • Block Explorer: https://basescan.org
4 Click "Save" and switch to Base network

Alternative: Coinbase Wallet

If you're using Coinbase Wallet, Base is already included:

  • Download Coinbase Wallet app
  • Create or import your wallet
  • Base network is automatically available
  • Easy bridge from Coinbase exchange

üí∏ Getting ETH on Base

Method 1: Bridge from Ethereum

Transfer ETH from Ethereum mainnet to Base:

  1. Visit Base Bridge: Go to bridge.base.org
  2. Connect Wallet: Connect your MetaMask or Coinbase Wallet
  3. Select Amount: Choose how much ETH to bridge
  4. Confirm Transaction: Pay gas fees on Ethereum
  5. Wait for Confirmation: Usually takes 10-20 minutes

Method 2: Buy Directly on Base

Purchase ETH directly on Base network:

  1. Use Coinbase: Buy ETH and withdraw to Base
  2. Use Ramp: Buy crypto with fiat directly on Base
  3. Use MoonPay: Another fiat-to-crypto option
  4. Use Transak: Credit card purchases on Base

Method 3: Swap from Other Tokens

If you have other tokens on Base, swap them for ETH:

  • Uniswap: Swap any token for ETH
  • SushiSwap: Alternative DEX on Base
  • 1inch: Aggregator for best prices
