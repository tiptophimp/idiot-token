Install MetaMask, connect to Base, and add IDIOT Token securely.

‚ö†Ô∏è Security Warning: Never share your seed phrase with anyone. MetaMask will never ask for it.

Step 1: Install MetaMask

Download and install the MetaMask browser extension:

  • Visit metamask.io
  • Click "Download" and install for your browser
  • Follow the installation prompts

Step 2: Create Wallet

Set up your new wallet securely:

  • Click "Create a new wallet"
  • Create a strong password
  • IMPORTANT: Write down your seed phrase and store it safely offline
  • Never take screenshots or store it digitally

Step 3: Add Base Network

Configure MetaMask for Base network:

  • Click the network dropdown (top of MetaMask)
  • Select "Add Network" or "Custom RPC"
  • Enter these details:
    • Network Name: Base
    • RPC URL: https://mainnet.base.org
    • Chain ID: 8453
    • Currency Symbol: ETH
    • Block Explorer: https://basescan.org
  • Click "Save"

Step 4: Import IDIOT Token

Add IDIOT Token to your wallet:

  • In MetaMask, click "Import tokens"
  • Paste the contract address: 0xC29EF04CFFe38012dcfc1E96a2B368443f298dE1
  • Token should auto-populate as "IDIOT"
  • Click "Add Custom Token"

Step 5: Practice with Small Amounts

Test your setup safely:

  • Send a small amount of ETH to your wallet
  • Try buying a small amount of IDIOT
  • Practice sending/receiving tokens
  • Get comfortable with the process
